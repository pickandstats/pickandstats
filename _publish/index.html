Qué hacemos

Estadística avanzada

Ratings ofensivo y defensivo, Four Factors, ritmo, eficiencia real de tiro, percentiles… las métricas que de verdad explican un partido, calculadas a partir de las actas oficiales.

Análisis de equipos y jugadores

Perfiles automáticos que traducen los números en lenguaje claro: fortalezas, debilidades y claves para preparar un partido. Scouting al alcance de cualquiera.

Fases de ascenso

El desenlace de cada temporada, contado como se merece: eliminatorias, liguillas y el camino de cada equipo hacia el ascenso, con el histórico de varias temporadas.

Guías que resuelven dudas

Cómo funcionan los ascensos, los descensos, el reglamento… las preguntas que se repiten cada año, explicadas con claridad y sin letra pequeña.

Las tres categorías FEB

El baloncesto de las categorías nacionales mueve a miles de jugadores y aficionados, pero apenas nadie le dedica datos serios. Pick&Stats nace para cambiarlo y hoy cubre Primera, Segunda y Tercera FEB: liga regular, fases de ascenso y descenso, y varias temporadas de histórico para poder comparar.
